#+title: Configuration

* Home Manager options

All options live under =services.stt-anywhere=:

| Option              | Type            | Default                        | Description                                                            |
|---------------------+-----------------+--------------------------------+------------------------------------------------------------------------|
| =enable=            | bool            |                                | Enable the stt-anywhere client service                                  |
| =enableServer=      | bool            | =true=                         | Run a local moshi-server. Set =false= for remote-only clients          |
| =cudaCapability=    | string          | ="8.6"=                        | CUDA compute capability for moshi-server                               |
| =serverAddr=        | string          | ="127.0.0.1"=                  | Bind address for moshi-server. ="0.0.0.0"= to accept remote clients   |
| =serverUrl=         | string          | =ws://127.0.0.1:${serverPort}= | WebSocket URL the client connects to. Override for remote servers      |
| =serverPort=        | port            | =8098=                         | Port for moshi-server                                                  |
| =sttModel=          | string          | ="kyutai/stt-1b-en_fr-candle"= | HuggingFace model repo                                                |
| =language=          | null or string  | =null=                         | Language hint (=null= for auto-detect)                                 |
| =extraEnvironment=  | attrs of string | ={}=                           | Extra env vars for the stt-anywhere service                             |
| =package=           | package         | /auto/                         | Override the stt-anywhere client package                                |
| =moshiPackage=      | package         | /auto/                         | Override the moshi-server package                                      |

* Environment variables

These can be set via =extraEnvironment= or in the shell when running standalone:

| Variable                | Default                | Description                          |
|-------------------------+------------------------+--------------------------------------|
| =STT_ANYWHERE_SERVER=    | =ws://127.0.0.1:8098=  | WebSocket URL for moshi-server       |
| =STT_ANYWHERE_API_KEY=   | =public_token=          | API key for moshi-server auth        |
| =STT_ANYWHERE_TARGET=    | /empty (default mic)/  | PipeWire node id/name to capture     |

* Keybinds

Send =SIGUSR1= to toggle recording, =SIGUSR2= to switch between mic and system audio capture.

** niri

#+begin_src kdl
binds {
    Mod+V       { spawn "systemctl" "--user" "kill" "-s" "USR1" "stt-anywhere.service"; }
    Mod+Shift+V { spawn "systemctl" "--user" "kill" "-s" "USR2" "stt-anywhere.service"; }
}
#+end_src
